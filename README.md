# Viewing the project 🔍


## Cloning 🧑‍🤝‍🧑

``` shell
$ cd ~/workspace
$ git clone https://github.com/escapingyouth/Portfolio.git
$ cd Portfolio
```

## Running the project 🏃‍♀️💨
``` shell
$ npm run start
```
Open http://localhost:3000 to view it in your browser.
